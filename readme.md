# prettier traversing `.git/` files

> For issue [#4895](https://github.com/prettier/prettier/issues/4895)

Steps to reproduce:

```shell
git clone https://github.com/gillchristian/prettier-report.git
cd prettier-report
yarn install
yarn format
```

_Or with_ `npm`:

```shell
git clone https://github.com/gillchristian/prettier-report.git
cd prettier-report
npm install
npm run format
```
